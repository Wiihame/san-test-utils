# san-test-utils

## 介绍

San Test Utils 是专门为 [San.js](https://baidu.github.io/san/) 而打造的测试库，它提供了各种为San组件进行单元测试的能力。

## 安装

```
npm install san-test-utils
```

## 测试

```
npm test
```
